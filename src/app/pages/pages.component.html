How To Use The Notes App

  1. This app isn't for your notetaking but referencing.

  2. Anything you're looking for, start with a keyword search using "ctrl F", or look for it in "Frequent Actions" below.

  3. Anything unclear, go into the code where that thing is implemented (you'll be doing this a lot). The comments there would help.
    (I often used the same names in the code as in the examples)

  4. You'll often find frames containing syntaxes which you can easily copy, paste & edit.
    (Use this app for reference purposes)

  5. The "Testing2" page is basically a free page, mainly for testing any random angular code you see online and want to try out.

Frequent Actions

I want to:
  1. get a button to click automatically: clickHere then go into the code of that page and see how it's done.
  2. display data items, all with the same formatting: clickHere to view the list, then go to the code of that page, ctrl+F '*ngFor', then see how it's done.
  3. filter a list by a specific criterion: clickHere then read it. If you need to check the code of the page..
  4. filter a list by keyword: clickHere then read it. If you need to, check the code of the page..
  5. route by id: clickHere then read it. You may also check the reference links if it's not clear..
  6. show data on the view, from the backend/ts file: clickHere and read the notes, you can also check the code where it's implemented in the "view-detail-page" of the testing page folder for more clarity.
  7. perform a post (create) operation: clickHere then read the notes, check the code where it's implemented in the "create-new" folder of the testing page folder for more clarity.
  8. perform an edit (put) operation: clickHere then read the notes, also check the code where it's implemented in the "edit-page" of the testing page folder for more clarity.
  9. perform a delete operation: clickHere then read the notes. You can also check how it's implemented in the code of the "testing-page".
  10. delete using a confirmation modal (by id): clickHere then read it. You may also check the reference links if it's not clear..
  11. duplicate a html element: clickHere then read the notes below it. You can also go into the code of that page to see how it's done in more detail..
  12. add bootstrap or fontawesome to my app: clickHere and read the notes. Then try it out yourself..
  13. display clicked items on a modal: clickHere then click the view button. It'll display the details of that item on a modal. You can then go into the code of that page to see how it's done..
  14. do data validation on a form/create rules for the user's filling of the form: clickHere then click the "New +" button. It'll show you a form on which validation has been added. Go into the code of that page and see how it is done..
  15. do pagination: clickHere and read the notes. You're most likely to use the "Pagination with pageSlice".
